# LifePocket Android 内测隐私说明

适用范围：LifePocket Android 小范围内测版。

LifePocket 用于把用户主动选择的截图、票据、账单、预约、购物、旅行和待办文本整理成生活卡片、提醒、消费记录和清单。

## 数据处理

- API Token：由用户在设置页自行填写，原生 Android 版本使用 `expo-secure-store` 保存在本机。
- 图片和文本：默认保存在本机；只有用户点击识别时，才会发送到当前配置的模型 API Endpoint。
- 生活卡片和清单：当前内测版使用本机存储，不做云同步。
- 通知提醒：仅用于在本机创建用户保存的生活提醒。

## 权限用途

- 相机：用于拍摄小票、账单和生活信息截图。
- 相册/图片读取：用于选择图片并进行 AI 识别。
- 通知：用于发送本地提醒。

## 用户注意事项

请不要把身份证、银行卡、医疗记录、完整住址、真实密钥等敏感信息提交给不可信的模型接口。使用者需要自行确认所配置模型 API 的数据处理规则。

## 内测限制

当前版本面向小范围测试，不承诺云备份、账号同步、数据恢复或商店级 SLA。卸载应用可能会清除本机数据。
